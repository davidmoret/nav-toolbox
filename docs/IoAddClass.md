## [@insite](../../README.md)/[nav-toolbox](../README.md)/IoAddClass.js

Observe le scroll et ajoute une classe lorsque l'élément apparait

* *@param {NodeList} elements* 
* *@param {Object} options* 
    * classToAdd: null
    * once: false
    * observerOptions
        * rootMargin: '-200px'
        * threshold: .01

#

#### Usage :


```bash
new IoAddClass({
    classToAdd: 'visible',
    once: true
});
```
